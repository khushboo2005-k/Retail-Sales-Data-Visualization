# Overview
This project is part of my Data Science Internship task. The goal of this project is to analyze sales data using Python and visualize trends using different types of charts
# Tools & Technologies

* Python 🐍
* Pandas 📊
* Matplotlib 📈
* Seaborn 🎨
* Google Colab

## 📊 Project Work

### 1. Line Chart (Sales Over Time)

* Shows how sales change over time
* Helps identify trends and patterns

### 2. Bar Chart (Category Comparison)

* Compares sales across categories
* Identifies the highest performing category

### 3. Pie Chart (Category Share)

* Displays percentage contribution of each category

---

## 📈 Results & Insights

* Sales show fluctuations over time with some peaks
* Technology category has the highest sales
* Furniture and Office Supplies follow behind
* Pie chart clearly shows category distribution
